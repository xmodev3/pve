# XMODE PVE

Утилита для быстрого прохождения спецопераций. Ниже предоставлены описание функций и бинды для переключения

## Системные требования

* Сборка винды 10 должна быть выше 1903
* На 11 винде до 23h2

## Функции

| Клавиша | Функция              | Состояние по умолчанию    |
| ------- | -------------------- | -----------------------  |
| `F1`    | Силуэты              |    Включено   |
| `F2`    | Хиты                 |    Включено   |
| `F3`    | Аим на ракеты        |    Включено   |
| `F4`    | Проверка видимости   |    Отключено  |
| `F5`    | Быстрая смена оружия |    Включено   |
| `F6`    | Экстра функции       |    Включено   |
| `F7`    | Автоактивация        |    Отключено  |
| `F8`    | Автопроход           |    Отключено  |
| `F9`    | Автостарт            |    Отключено  |
| `F10`   | Автопрыжок           |    Отключено  |

### Экстра функции

> Работают только при включённом `F6`

| Клавиша    | Функция                                                 |
| ---------- | ------------------------------------------------------- |
| `Insert`   | Убийство команды                                        |
| `Delete`   | Убийство себя                                           |
| `Z`        | Жираф                                                   |
| `T`        | Притяжение к богомолу / Наводка на вертолёт с предиктом |
| `PageDown` | Скрытие ников                                           |
| `E`        | Поднятие оружия на увеличенном расстоянии               |
| `N`        | Телепорт щита на восходе                                |

### Постоянно работающие функции

* Анти-афк
* Автопроход кампании
* Автоперезарядка
* Спам в глобальном чате
* При включенном F6: Авто-подбор оружия

## Описание функций

### `F2` — Хиты

Все специфические цели для боссов работают автоматически — ничего переключать не требуется.

### `F3` — Аим на ракеты

Автоматически стреляет из гранатомета из инвентаря в цель, предсказывает позицию цели.

Работает на:

* все существующие вертолёты на текущих спецоперациях,
* Гром и Накаи *(Ночной Город)*,
* корабли *(Ледокол)*,
* некоторые стационарные вышки автоматически
  *(например, на тренировке)*.

![](https://github.com/xmodev3/pve/blob/main/gif/ship1.gif?raw=true)

### `F7` — Автоактивация

* Автовоскрешение
* Автолаз по верёвке
* Автоактивация

### `F8` — Автопроход

Автоматически играет за вас.
Работает только на некоторых спецоперациях.

* Тренировка
* Анубис
* Вулкан
* Опасный эксперимент
* Белая акула
* Чёрная акула
* Снежный бастион *(частично)*
* Затмение

### `F9` — Автостарт

* запускает комнату,
* пропускает результаты,
* нажимает «Готов».

### `F10` — Автопрыжок

* Синхронизирован с хитами.
* Прыгает прямо перед выстрелами.
* Не прыгает лишний раз, если рядом нет целей.
* Прыгает ради получения очков.

### Анти-АФК

Работает постоянно автоматически.

### Автопроход кампании

Работает в фоне автоматически, когда игрок находится:

* в лобби,
* на складе,
* на полигоне,
* на канале PVE.

> В бою функция не работает.
> Полный проход кампании занимает чуть больше 4 часов, зависит от миссии, к примеру проход Пролога занимает 3 минуты

### Автоперезарядка

* Работает в фоне.
* Не мешает игровому процессу.
* Незаметно перезаряжает оружие в инвентаре.
* Автоматически использует комплект с патронами при их окончании.

Постоянно отправляет сообщения в глобальный чат.
